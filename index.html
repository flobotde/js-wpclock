js-wpclock

Description

js-wpclock can be used to display a clock as background image for a HTML website.

It uses the images that can be unzipped from wallpaperclock (.wcz) files which can be downloaded for free from http://www.vladstudio.com/wallpaperclock or can be designed by yourself (http://www.vladstudio.com/wallpaperclock_create/).

Wallpaper clock images can include information like current time, date, weekday, zodiac sign and moonphase, depending on availability in wcz file.

Requirements

  • CSS3 compatible browser
  • jQuery (http://jquery.com/)

Use

Download a .wcz package and unzip it and place the images in a directory (eg /img/) where your images for your webpage reside, or create your own clock images.

In your HTML Head Section load the needed JavaScript Files, eg.: 

      
         <script src="js/js-wpclock.js"></script>
         <script src="http://code.jquery.com/jquery-1.7.2.min.js"></script>
and start the clock with the following code 
      
         <script>
           jQuery(document).ready(function($) {
             startClock($,'../img/','analog');
            });
         </script>
Parameters:
  1. $ is always jQuery
  2. Directory Path where the clock images are, relative to the JS-File
  3. boolean: is the clock digital/analog?
    digital: has hour0.png to hour23.png
    analog: has hour0.png to hour59.png
